# 🚀 Git Learning Repository

This repository contains my Git learning journey from basic to advanced concepts with practical examples, commands, workflows, and interview questions.

## 📚 Topics Covered

### 1. Git Basics
- What is Git?
- Git vs GitHub
- Git Installation
- Configure Git
- Git Architecture

### 2. Repository
- git init
- git clone
- git status

### 3. Staging Area
- git add
- git restore
- git rm

### 4. Commit
- git commit
- git commit -m
- git commit --amend

### 5. Branching
- Create Branch
- Switch Branch
- Delete Branch
- Rename Branch

Commands:
```bash
git branch
git switch
git checkout
```

### 6. Merge
- Fast Forward Merge
- Three-Way Merge
- Merge Conflict
- Merge Workflow

Commands:
```bash
git merge <branch-name>
```

### 7. Diff
- Compare Working Directory
- Compare Staging Area
- Compare Commits
- Compare Branches

Commands:
```bash
git diff
git diff main feature/login
git diff main...feature/login
```

### 8. Reset
- Soft Reset
- Mixed Reset
- Hard Reset

Commands:
```bash
git reset --soft HEAD~1
git reset --mixed HEAD~1
git reset --hard HEAD~1
```

### 9. Revert
- Undo Pushed Commit
- Safe History Management

Commands:
```bash
git revert HEAD
git revert <commit-id>
```

### 10. Logs
```bash
git log
git log --oneline
git log --graph
git log --decorate
```

### 11. Remote Repository
- git remote
- git fetch
- git pull
- git push

### 12. GitHub Workflow
- Create Repository
- Push Code
- Clone Repository
- Pull Request
- Merge Pull Request

### 13. Best Practices
- Write meaningful commit messages
- Create feature branches
- Keep commits small
- Never push secrets
- Use .gitignore
- Pull before push
- Avoid force push on shared branches
